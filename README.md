## 배포 주소
https://yuna-library.vercel.app/

## 시작 가이드
### Requirements
For building and running the application you need: 
- Node.js 18.12.1
- Pnpm 8.6.3
  

### Installation
```
$ git clone https://github.com/Cho-yunah/my_app.git
$ cd my_app
$ nvm use v.18.12
$ pnpm run dev
```
